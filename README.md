## 🎯 Sobre o Repositório

Este repositório reúne meus **estudos aprofundados e projetos práticos desenvolvidos em JavaScript**. O objetivo é consolidar o domínio sobre a linguagem, explorando desde conceitos fundamentais até técnicas avançadas de manipulação do DOM, programação assíncrona e arquitetura de objetos.

O conteúdo abrange:
- **Fundamentos Sólidos**: Variáveis, tipos de dados e lógica.
- **Ecossistema Web**: Manipulação extensiva do DOM e efeitos visuais.
- **Modern JavaScript**: Classes, Factory Functions, e ES6+.
- **Async JS e Automação**: Consumo de APIs e ferramentas de desenvolvimento front-end.

<br />

## 🚀 Como Executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/jrs-neto/javascript.git
   ```

2. **Navegue até a pasta desejada:**
   Cada diretório possui seus próprios arquivos `index.html` ou scripts `.js` que podem ser executados diretamente no navegador ou via Node.js.

3. **Para módulos com automação:**
   Navegue até `automacao-front-end`, instale as dependências e execute o script de dev:
   ```bash
   npm install
   npm run dev
   ```
